Firebase is a platform for mobile application and web development, including simple and powerful APIs without the need for a backend or server.

Pros and cons of Firebase

Pros:

Enable using API with other problems
Really helpful when your apps runs on a centralized database
Stored in the cloud so readily available everywhere.
Data hosting is not an issue (. -Meaning if you are storing a lot of data, you don't have to worry about hardware!)

Cons

Level of reporting does not match that SQL
The app doesn’t work well if database updation is not frequent
Migration of apps can be a challenging task costs
